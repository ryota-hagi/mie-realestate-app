# 注文住宅比較.com SEO戦略

## SEOの基本方針（2026年3月策定）

### 勝てない領域（リソースを割かない）

大手HM（ダイワハウス・トヨタホーム・セキスイハイム等）のブランドKWは、SUUMO・HOME4U・価格.comなどDA90+の巨大サイトが独占しており、コンテンツの質に関係なく上位は取れない。以下のようなKWでの記事作成・最適化は費用対効果が低い:

- `ダイワハウス 坪単価` → 現状73位、改善不可能
- `トヨタホーム メリット` → 現状62位、同上
- `セキスイハイム 坪単価 比較` → 現状90位、同上

### 勝てる領域（集中投下する）

#### 1. 三重県のローカルビルダー・工務店・設計事務所を幅広くカバー

三重県で活動するローカルな住宅会社は、大手と違って競合記事がほとんどない。
「○○（会社名） 坪単価」「○○ 口コミ」といったKWは検索ボリュームは小さいが、
ほぼ無競合で1ページ目を取れるブルーオーシャン。

**方針**: 既存15社に絞らず、三重県北部で実際に施工している工務店・設計事務所・
ビルダーを幅広く発掘して記事化していく。1社1記事ではなく、まとめ記事や
比較記事の形でもよい。

**狙うKWの例**:
- `三重県 工務店 おすすめ` / `三重県 工務店 ランキング`
- `四日市 工務店 評判` / `桑名 工務店 おすすめ`
- `三重県 設計事務所 注文住宅` / `三重 建築家 住宅`
- `三重県 ローコスト住宅 工務店`
- `（個別の地方工務店名） 坪単価` / `（同） 評判`

**記事の切り口**:
- エリア別おすすめ工務店まとめ（四日市編・桑名編 等）
- 価格帯別の工務店比較（坪40万円台・50万円台・60万円台）
- 「自然素材が得意な工務店5選」「デザイン住宅が得意な設計事務所」等の特徴別まとめ
- 個別の地方ビルダー紹介記事（坪単価・施工事例・口コミ）

#### 2. 「三重県 × 具体テーマ」のロングテールKW

三重県に特化したKWは大手メディアがカバーしておらず、上位が取りやすい。

**成功事例**: `三重県 住宅補助金 2026` → **3.1位・CTR 5.66%**（唯一クリックされている記事）

**同じパターンで狙うべきKW群**:

##### 都市別の注文住宅KW（都市ガイド記事で対応）
- `四日市 注文住宅 おすすめ` / `四日市 土地 相場`
- `桑名 注文住宅 費用` / `桑名 土地探し`
- `鈴鹿 注文住宅 ハウスメーカー` / `鈴鹿 土地 坪単価`
- `いなべ 移住 注文住宅` / `いなべ 土地 安い`
- `亀山 注文住宅` / `亀山 土地 相場`
- `菰野 注文住宅` / `菰野 移住`（★記事未作成）
- `東員 注文住宅` / `東員 子育て`（★記事未作成）

##### 三重県×テーマの組み合わせKW
- `三重県 注文住宅 相場` (現38位→20位以内を目指す)
- `三重県 注文住宅 補助金` (成功事例、維持・強化)
- `三重県 住みやすい街` / `三重県 子育て しやすい 市`
- `三重県 注文住宅 ローコスト`（★記事未作成）
- `三重県 工務店 ランキング`（★記事未作成）
- `名古屋 通勤 三重`（現4.5位、強化すべき）
- `三重県 学区 人気`（現7.3位、強化すべき）

### 記事作成の優先順位ルール

新しい記事を作るときは、以下の優先度で判断する:

1. **最優先**: 三重県ローカルの工務店・設計事務所の紹介・比較記事（競合ほぼゼロ）
2. **高**: 「三重県 × 具体テーマ」のロングテールKW記事
3. **高**: GSCで表示があるが順位20位以下のKWをカバーする記事
4. **中**: 都市ガイド記事の未作成分（菰野・東員）
5. **低**: 一般的な注文住宅ノウハウ記事（競合が多すぎる）
6. **やらない**: 大手HMのブランドKW記事の新規作成

### KPIと効果測定

- **短期（1-2週間）**: タイトル改善済みページのCTR変化
- **中期（1-3ヶ月）**: 三重県ローカルKWの順位改善（20位→10位以内）
- **長期（3-6ヶ月）**: 月間クリック数50以上を目指す
