# サイトアクセス増加戦略（SEO以外）

> サイトへのトラフィックを増やすことに特化した施策集。

---

## 現状の課題

| 項目 | 現状 |
|------|------|
| 流入経路 | SEOのみ。他チャネルゼロ |
| SNS | 未運用（OGPタグ設定済み、シェアボタンなし） |
| 広告 | 未実施 |
| 外部連携 | なし |
| バイラル設計 | なし（シェア機能未実装） |
| データ資産 | 国交省取引18,484件、学区GeoJSON、7市町詳細データ（未活用） |

### なぜアクセスが増えないのか

1. 流入チャネルがSEO一本 — 順位が上がらなければゼロ
2. サイトの存在を知る手段がない — SNS・広告・口コミ・外部リンクいずれもなし
3. バイラルの仕組みがない — シェアボタンなし、シェアしたくなる仕掛けなし
4. リピート訪問の仕組みがない — 一度来た人を呼び戻す手段がない

```

「坪単価8万円台」のいなべ市
```

```
③ 諸費用（10%前後）

「坪単価が安い＝総額が安い」とは限らない → [URL]
```

**ツール紹介:**

### 運用ポイント

- スレッドで「7市町を1つずつ解説」等の連投
