## 戦略1: コンテンツSEO（検索流入の量産）

**最優先。検索からの安定的な流入基盤を作る。**

### 1-1. ロングテールキーワード記事の量産

ナレッジセクション (`/knowledge/`) に記事を追加していく。
1記事1キーワードを狙い、月4〜8記事ペースで増やす。

### 1-2. 既存記事の拡充

現在のナレッジ5記事は文量が薄い可能性。各記事を2,000〜4,000字に拡充し、
図解・表・チェックリストを追加して「このページで完結する」品質にする。

### 1-3. 内部リンク戦略

```
ナレッジ記事 → エリアページへ誘導（「実際の相場を見てみる →」）
エリアページ → 関連ナレッジへ誘導（「土地選びのコツはこちら」）
全ページ → 物件比較ツールへCTA（「物件を比較してみる」）
```

コンテンツクラスターを形成し、サイト内回遊を高める。

## 戦略2: SNS運用（拡散とブランド認知）

### 2-1. X (Twitter) 運用

**アカウント方針**: 「三重県で家を建てる人の味方」ポジション

**投稿パターン（週5投稿目安）**:
| 曜日 | 内容 | 例 |
|------|------|-----|
| 月 | 相場データ | 「四日市市の住宅地 坪単価は平均○万円。5年前と比べると...」 |
| 水 | ナレッジ記事宣伝 | 「注文住宅の費用、内訳を知らないと100万単位で損します → 記事リンク」 |
| 金 | ツール紹介 | 「SUUMOの物件URL貼るだけで比較表が作れるツール、知ってる？」 |
| 随時 | 不動産ニュース | 三重県の地価発表、金利変動、制度変更に即反応 |
| 随時 | ユーザーの声 | 使ってくれた人のリポスト、感想引用 |

**ハッシュタグ例**:
`#注文住宅` `#三重県` `#四日市` `#土地探し` `#マイホーム計画` `#家づくり`

### 2-2. Instagram 運用（中期）

**カルーセル投稿**でナレッジ記事を視覚化:
- 「注文住宅の費用内訳」→ 5枚スライドの図解
- 「三重県の住みやすい街TOP5」→ エリア写真+データ
- 「土地選びチェックリスト10項目」→ 保存したくなるデザイン

リール動画:
- 「坪単価の見方、30秒で解説」
- 「この土地、買っていい？AIに聞いてみた」

## 戦略5: サイト改善による回遊・再訪促進

### 5-1. 導線改善

- **全ページ共通CTAバー**: 「物件を比較してみる」「エリアを比較する」を常時表示
- **関連記事レコメンド**: ナレッジ記事の末尾に「あわせて読みたい」セクション
- **パンくずリスト**: 全ページに設置（ナビゲーション + SEO効果）

### 5-2. リピート施策

- **お気に入り物件の保存機能**: ログインユーザーが物件を保存→再訪動機
- **比較結果のシェア機能**: URLで比較結果を共有可能にする→SNS拡散
- **メールマガジン**: 「三重県の相場レポート（月1回）」の配信
  - トップページにメール登録フォームを設置
  - Supabase + Edge Functionで配信基盤構築

### 5-3. UGC (ユーザー生成コンテンツ)

- **物件レビュー・口コミ機能**: エリアページに「住んでる人の声」セクション
- **比較事例の公開**: 匿名化した比較結果を「みんなの比較」として掲載
