# 三重県注文住宅・土地探しサイト 集客戦略（SEO以外）

## サイト現状分析

| 項目 | 現状 |
|------|------|
| 対象エリア | 三重県北部7市町（四日市・桑名・鈴鹿・いなべ・亀山・菰野・東員） |
| ターゲット | 注文住宅を検討中の土地購入希望者 |
| 主要機能 | 物件比較ツール、エリア比較、コストシミュレーター、ナレッジ記事 |
| 収益モデル | 未確立（全機能無料提供） |
| リード獲得 | 未整備（問い合わせフォーム準備中、メール登録なし） |
| 認証 | Supabase匿名認証（メールアドレス不要） |
| OGP | 設定済み（SNSシェア対応） |

### 現状の根本的課題

1. **リード獲得導線がない** — ユーザーが来訪しても連絡先を取得する仕組みがない
2. **流入経路がSEOのみ** — 他チャネルからの流入が設計されていない
3. **リピート接点がない** — 再訪を促す仕組み（メール・LINE等）が未整備

## 戦略6（最優先）: リード獲得基盤の整備

## 戦略1: SNS運用（Instagram / X / YouTube）

### Instagram（最重視）

| 項目 | 内容 |
|------|------|
| 理由 | 住宅検討層（30代夫婦）の情報収集チャネルとして最も利用されている |
| アカウント設計 | 「三重県の土地探し専門アカウント」としてブランディング |
| 投稿頻度 | 週3-4回 |

**コンテンツ案:**

| 種類 | 内容例 | 目的 |
|------|--------|------|
| リール動画 | 「四日市の坪単価、実は○○万円」「この土地、ハザードマップで見ると…」 | リーチ拡大 |
| カルーセル投稿 | 「土地選び10のチェックリスト」「建蔽率・容積率を3分で理解」 | 保存・シェア |
| ストーリーズ | 「今週の新着物件」「エリア比較クイズ」 | エンゲージメント |
| まとめ投稿 | 「桑名市vs四日市市 住みやすさ比較」 | サイト誘導 |

**運用のポイント:**
- プロフィールリンクにサイトURLを設置（linktree等でLINE・サイト・予約を集約）
- 「土地探し中の方はプロフィールの無料ツールへ」をCTAとして統一
- ハッシュタグ: `#三重県注文住宅` `#三重県土地探し` `#四日市マイホーム` `#桑名市新築` 等
- 既存のサイトデータ（取引事例・相場情報）を視覚的にリデザインして投稿に活用

## 戦略2: 地域連携・パートナーシップ

三重県北部という限定エリアのため、地域密着の連携が非常に有効。

### 2-1. 工務店・ハウスメーカーとの連携

| 項目 | 内容 |
|------|------|
| 形態 | 土地探しツールの無料提供 → 相互送客 |
| メリット（先方） | 「土地がまだ決まっていない」見込み客への対応ツールとして活用できる |
| メリット（自社） | 工務店の顧客にサイトを紹介してもらえる、信頼性向上 |
| アプローチ | 三重県北部の中小工務店10-20社にDM・訪問営業 |
| 具体施策 | 工務店専用の比較レポートページ作成（共同ブランディング） |

| 施策 | FP相談窓口にチラシ設置、相互紹介制度の構築 |

### 2-3. 地域メディア・フリーペーパー

## サイト内改修の提案（集客施策を活かすために必要な改修）

現在のサイトは**ツールとしての完成度は高い**が、集客施策と連動させるには以下の改修が必要。

### 必須改修

1. **LINE友だち追加ボタンの設置**
   - 全ページ共通のフローティングボタン or フッター固定バナー
   - 物件比較結果表示後に「LINEで結果を受け取る」ボタン

2. **問い合わせ・相談予約フォームの実装**
   - `/about/` の「お問い合わせ準備中」を実際のフォームに置換
   - 比較結果画面・ナレッジ記事末尾にCTAバナー設置

3. **OGP画像の最適化**
   - SNSシェア時にクリックされやすい画像に更新
   - エリア別ページごとに個別OGP画像を設定

### 推奨改修

4. **ユーザー登録機能の強化**
   - 匿名認証 → メール or LINE認証への移行促進
   - 「比較結果を保存するにはログイン」の導線

5. **シェア機能の強化**
   - 比較結果のSNSシェアボタン（現在未実装）
   - 「この比較結果を友人に送る」機能

6. **GA4カスタムイベントの設定**
   - 物件比較実行、エリアページ閲覧、ナレッジ記事完読等のイベントトラッキング
   - 広告効果測定のためのコンバージョン設定

## まとめ

最も重要なのは**戦略6（リード獲得基盤）を最初に整備すること**。LINE公式アカウントと相談予約フォームがなければ、どれだけ集客しても「ザルに水」状態になる。

その上で、**Instagram運用（戦略1）と工務店連携（戦略2）**が、コスト対効果の観点から最も優先度が高い。三重県北部という限定エリアであるため、地域密着の施策がデジタル広告以上に効果を発揮する可能性が高い。

Google広告（戦略3）は即効性があるが、リード獲得の受け皿が整った後に開始すべき。
